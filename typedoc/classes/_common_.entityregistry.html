Class EntityRegistry

The EntityRegistry is the core register for all entities that are annotated with the entity decorators:

  • @Model
  • @Controller
  • @Seeder
  • @Migration
  • @Store
  • @Service

Both the static classes and any associated metatadata are assigned to the registry by the decorators
